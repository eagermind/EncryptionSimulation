# EncryptionSimulation
基于国密算法的采集终端链路加密仿真研究</br>

基于国密算法的采集终端链路加密仿真平台分为三部分：基于Python的嵌入式管理平台、
电网安全接入平台Socket服务端、电网安全接入平台WEB管理端。
整个采集终端链路实现了数据加密传输与签名认证。<br>
